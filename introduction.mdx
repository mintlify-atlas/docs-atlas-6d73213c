STM32CubeH7 is STMicroelectronics' comprehensive firmware package for the STM32H7 microcontroller series. It bundles everything you need to build embedded applications — from low-level CMSIS core support and hardware abstraction layer (HAL) drivers, to board support packages (BSP), and a complete set of third-party middleware libraries including FreeRTOS, LwIP, FatFs, and USB stacks.

## Supported boards

STM32CubeH7 targets 21 official ST boards across the STM32H7 portfolio:

- **Nucleo boards**: NUCLEO-H743ZI, NUCLEO-H753ZI, NUCLEO-H723ZG, NUCLEO-H745ZI-Q, NUCLEO-H755ZI-Q, NUCLEO-H7A3ZI-Q
- **Discovery kits**: STM32H745I-DISCO, STM32H747I-DISCO, STM32H750B-DK, STM32H735G-DK, STM32H7B3I-DK
- **Evaluation boards**: STM32H743I-EVAL, STM32H753I-EVAL, STM32H747I-EVAL, STM32H757I-EVAL, STM32H7B3I-EVAL

## Project types

For each supported board, STM32CubeH7 includes:

| Type | Description |
|------|-------------|
| **Examples** | Single-peripheral demonstrations (GPIO, UART, ADC, TIM, DMA, etc.) |
| **Examples\_LL** | Same coverage using the low-layer LL driver API |
| **Examples\_MIX** | Mixed HAL/LL usage patterns |
| **Applications** | Multi-peripheral applications using middleware (FreeRTOS, USB, LwIP, FatFs) |
| **Demonstrations** | Full-featured demo firmware showcasing board capabilities |
| **Templates** | Minimal project skeletons to start your own application |

<Tip>
  Use **Templates** as the starting point for new projects. They include the correct clock configuration, linker scripts, and startup files for your target board.
</Tip>
